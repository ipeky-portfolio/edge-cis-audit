# Risikobewertung

## Hoch

- Safe Browsing deaktiviert
- HTTPS-only deaktiviert
- Unsichere Extensions installiert

## Mittel

- Passwortspeicherung / Autofill aktiviert
- Pop-ups / Downloads zu locker konfiguriert
- Telemetrie aktiv

## Niedrig

- Optionale Hardening-Einstellungen nicht umgesetzt

## Compliance Score

(Anzahl konformer Controls / Gesamtzahl anwendbarer Controls) x 100%
